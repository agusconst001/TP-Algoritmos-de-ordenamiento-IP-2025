Consigna

El objetivo de este proyecto es crear un visualizador interactivo que muestre cómo funcionan los distintos algoritmos de ordenamiento paso a paso. La aplicación permite observar visualmente cómo se reordenan los datos, ayudando a comprender mejor sus diferencias y eficiencias.

Algoritmos Trabajados

  • Bubble Sort
  • Selection Sort
  • Insertion Sort
  • Quick Sort
  • Shell Sort
